DESCRIPTION:
This program contains Mycoplasma, ChromaCell, Evolver, DiseasedCells, Predator & Prey cells.

USER INSTRUCTIONS:
The simulator method can be updated to add method for populating new life form.
The populate() method calls necessary populate method for specified life forms, only 1 should be called at a time except for Predator & Prey.
Predator and prey are to be run in tangent, and should both be called in the populate() method when testing their simulation.
To run, create new JavaFX from SimulatorView class, then run method simulate(int n) to run n generations in simulator.
